# GTresD — Landing Page

Landing page minimalista tipo "link en bio" para **GTresD**, servicio de impresión 3D personalizada en Merlo, Buenos Aires, Argentina.

##  Demo

Ver proyecto en vivo → **[GTresD Calculator](https://jgastonmurua.github.io/GTresD/)**

## Stack

- HTML5 + CSS3 vanilla — sin frameworks
- Google Fonts: Orbitron + Exo 2
- SVG inline para íconos (sin dependencias externas)

## Características

- SEO local optimizado (meta tags, Schema.org LocalBusiness, Open Graph)
- Diseño mobile-first, 100% responsive
- Animaciones CSS ligeras (fade, glow, shimmer)
- Botón WhatsApp con mensaje preconfigurado
- Links a Instagram, Facebook y TikTok

## Personalización rápida

| Qué cambiar | Dónde |
|---|---|
| Foto de perfil | Clase `.profile-pic` → reemplazá el SVG por `<img>` |
| Logo | Agregá `<img src="logo.png">` arriba del `.brand-name` |
| Número de WhatsApp | `wa.me/541130484275` |
| URL canónica | `<link rel="canonical">` y JSON-LD |
